

## Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/miatresenio/tictactoe-rn-expo.git
   
   ```

   Navigate to the project folder:
   cd tictactoe

   Install dependencies:
   npm install

   Run the app:
   npx expo start
